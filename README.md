## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A web browser
- [NPM](https://webpack.js.org/)
- [Webpack configured](https://webpack.js.org/)

### Setup

- Clone the project from Github

```git clone repolink```


### Install
